2. Add ocr_engine field to ConversionOptions dataclass after ocr_mode:
```python
ocr_engine: str = "auto"  # "auto" | "easyocr" | "rapidocr" | "tesseract"
```
Update docstring to mention ocr_engine.

3. Add the engine factory dict and install hints dict ABOVE build_pipeline_options():
```python
_OCR_ENGINE_CLASSES = {
    "auto":      OcrAutoOptions,
    "easyocr":   EasyOcrOptions,
    "rapidocr":  RapidOcrOptions,
    "tesseract": TesseractOcrOptions,
}

_ENGINE_INSTALL_HINTS = {
    "rapidocr":  "pip install rapidocr-onnxruntime",
    "tesseract": "apt-get install tesseract-ocr libtesseract-dev libleptonica-dev && pip install tesserocr",
}
```

4. Replace build_pipeline_options() body with engine factory logic:
- If ocr_mode == "off": return as before (no engine needed)
- Otherwise: look up opts.ocr_engine in _OCR_ENGINE_CLASSES; raise ValueError if unknown
- Instantiate engine_cls(force_full_page_ocr=force_full_page_ocr)
- Set ocr_options.lang = opts.ocr_languages ONLY when opts.ocr_engine == "easyocr" and opts.ocr_languages is non-empty
- Return PdfPipelineOptions(do_ocr=True, do_table_structure=opts.table_detection, ocr_options=ocr_options)

5. In DoclingAdapter.convert_file(), wrap DocumentConverter construction in try/except:
```python
try:
    converter = DocumentConverter(
        format_options={"application/pdf": PdfFormatOption(pipeline_options=pipeline_options)}
    )
except Exception as exc:
    hint = _ENGINE_INSTALL_HINTS.get(opts.ocr_engine, "the required OCR package")
    raise RuntimeError(
        f"OCR engine '{opts.ocr_engine}' is not available. Install: {hint}"
    ) from exc
```
The converter construction currently happens before the async with self._lock block — keep it there, just wrap it.

Anti-patterns:
- Do NOT set ocr_options.lang for tesseract or rapidocr — language format is incompatible
- Do NOT use TesseractCliOcrOptions (excluded per user decision)
- Do NOT silently fall back to a different engine if the requested one is missing
