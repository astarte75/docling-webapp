  <files>backend/job_store.py</files>
  <behavior>
    - Job dataclass has fields: job_id (str), tmp_path (str), events (asyncio.Queue, default_factory), status (str, default "queued")
    - conversion_worker is an async coroutine that loops forever (while True)
    - Worker sets job.status = "converting" and pushes {"type": "started", "message": "Conversion started"} before calling convert_file
    - Worker pushes {"type": "completed", "markdown": "..."} on success and sets status = "completed"
    - Worker catches RuntimeError and pushes {"type": "failed", "message": str(exc)}, status = "failed"
    - Worker catches bare Exception, logs it, pushes {"type": "failed", "message": "Internal server error"}, status = "failed"
    - Worker's finally block always deletes the temp file via os.remove if it exists
    - Worker calls state.dispatch_queue.task_done() at the end of each loop iteration
    - No terminal event is ever skipped — the SSE stream must always terminate
  </behavior>
  <action>
    Create `backend/job_store.py` with the following content:

    1. Imports: `asyncio`, `dataclasses` (dataclass, field), `logging`, `os`, and a TYPE_CHECKING block for DoclingAdapter (to avoid circular imports — import DoclingAdapter inside the function or use Any for the state type).

    2. `Job` dataclass:
       ```python
       @dataclass
       class Job:
           job_id: str
           tmp_path: str
           events: asyncio.Queue = field(default_factory=asyncio.Queue)
           status: str = "queued"
       ```

    3. `conversion_worker(state)` async coroutine:
       - `while True:` loop
       - `job_id = await state.dispatch_queue.get()`
       - `job = state.jobs[job_id]`
       - try block: set status, push started event, call `await state.converter.convert_file(job.tmp_path)`, push completed event with markdown
       - except RuntimeError: push failed event with str(exc), set status
       - except Exception: log with logger.exception, push failed event with "Internal server error", set status
       - finally: delete temp file if os.path.exists, call state.dispatch_queue.task_done()

    Use `logging.getLogger(__name__)` for the module logger.

    CRITICAL: The finally block must call `state.dispatch_queue.task_done()` — this enables callers to use `await dispatch_queue.join()` for testing. Without it the queue never marks items done.

    CRITICAL: Do NOT delete the temp file in the POST handler — only the worker deletes it (Pitfall 3 from research).
  </action>

       ```python
       @asynccontextmanager
       async def lifespan(app: FastAPI):
           logger.info("Starting up: initializing DoclingAdapter...")
           app.state.converter = DoclingAdapter()
           app.state.dispatch_queue = asyncio.Queue()
           app.state.jobs: dict[str, Job] = {}
           # Keep task reference — do not use fire-and-forget (GC would collect it)
           app.state._worker = asyncio.create_task(conversion_worker(app.state))
           logger.info("DoclingAdapter ready — application startup complete")
           yield
           # Graceful shutdown: cancel worker
           app.state._worker.cancel()
           try:
               await app.state._worker
           except asyncio.CancelledError:
               pass
           logger.info("Shutting down")
       ```
