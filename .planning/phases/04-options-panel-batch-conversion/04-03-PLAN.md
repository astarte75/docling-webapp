```typescript
// Importa dal file shadcn generato (verifica esatto path):
```

Page range inputs:
- Type="number", min="1"
- onChange: converte a `number | null` (se vuoto → null, non NaN)
- "to" vuoto = null (NON errore di validazione, come da decisione utente)
- `pageFrom: e.target.value ? parseInt(e.target.value) : null`

Switch table detection:
- `checked={value.tableDetection}`
- `onCheckedChange={(checked) => onChange({ ...value, tableDetection: checked })}`

Select OCR language:
- Valore singolo (non multi-select per semplicità)
- Se selezionato, `ocrLanguages: [selectedCode]`; se deselezionato/reset, `ocrLanguages: []`
- Aggiungi opzione "Qualsiasi (default)" come valore vuoto che imposta ocrLanguages=[]
